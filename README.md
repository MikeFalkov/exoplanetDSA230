# Exoplanet DSA-230

## Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/MikeFalkov/exoplanetDSA230.git
   ```

2. Navigate to the project directory:
   ```bash
   cd exoplanetDSA230
   ```

3. Synchronize dependencies:
   ```bash
   uv sync
   ```
